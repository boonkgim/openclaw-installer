# OpenClaw Installer

One-click installer for [OpenClaw](https://openclaw.ai).

## Download

Go to the [Releases](https://github.com/boonkgim/openclaw-installer/releases) page and download the installer for your platform:

- **macOS** — `OpenClaw Installer-1.0.0-universal.dmg`
- **Windows** — `OpenClaw Installer Setup 1.0.0.exe`

## Installation

### macOS
1. Download the `.dmg` file
2. Open it and drag the app to your Applications folder
3. Launch OpenClaw Installer from Applications

### Windows
1. Download the `.exe` file
2. Run the installer and follow the prompts
3. Launch OpenClaw Installer from the Start menu
